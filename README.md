# TTT: Translators' Training Tool
## Machine translation made easy for human translators!


### Authors
- Paula Estrella
- Roxana Lafuente
- Miguel Lemos


### Features
- Portable (Windows / Linux)


### Dependencies
- MOSES (Install with "--with-mm" and "--install-scripts" flags)
- Cygwin (only on Windows)
- GTK 3.0


### How to use
```
python main.py
```
